Направление проекта: "оценка стоимости автомобиля".

Описание проекта: создать программу (GUI-приложение), которая будет принимать от пользователя все необходимые данные автомобиля для расчета ее стоимости.

Каким образом программа будет знать рыночную стоимость автомобиля?: предварительно программа будет обучена, в принципе при каждом ее запуске она будет обучаться на основании составленной базы данных, взятых из интернета (возможно готовый прайс-лист, либо парсинг).

Количество обучаемых данных: не менее 30 позиций (постараюсь около 100-150 поз. добавить для точности вычисления).

Источник: все данные планирую брать с сайта auto.ru.

Почему именно автомобили?: большое количество параметров, от которых напрямую зависит стоимость автомобиля, надеюсь, что удастся предсказать стоимость в пределах 2-5%.

Тестовые данные: для тестирования валидности данной программы составлю данные с этого же сайта в количестве 10-20 позиций.

Способ обучения: с помощью MLPClassifier.

Визуализация: прилагаю предполагаемый интерфейс.
